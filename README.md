# React + Vite + D3 Skeleton

This is a minimal React+Vite+TypeScript skeleton with a D3 graph on the left (75%) and a sidebar on the right (25%).

## Run

```bash
npm install
npm run dev
```

## Layout

- Left: `Graph` component renders an SVG force-directed graph with D3.
- Right: `Sidebar` component is a placeholder for controls.
